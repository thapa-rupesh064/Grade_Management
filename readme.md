A simple, menu-driven CLI to record and analyze student grades.


## Overview
Grade Manager is a small Python program (single-file CLI) that lets you add student grades, view all grades, and compute the average grade per student using an interactive menu.

---

## Features
- Add a grade for a student (creates a new student entry when needed)
- View all student grades
- Compute and print the average grade for each student
- Small and dependency-free (pure Python)

---

## Requirements
- Python 3.8+ (works on Python 3.x)

## How to run this program
1. Open a terminal in the project folder
2. Run:

```bash
python3 main.py
```
